What is Ursine?

Ursine is a financial sentiment analysis tool built with Python. It leverages the power of the distilBERT model which was fine-tuned on financial news data, to deliver precise sentiment insights. Ursine offers a solution to generalize the sentiment behind financial articles, enabling smarter and more educated financial decisions.

Text Analysis

This tool is used to determine the sentiment of financial text with a relatively high degree of accurate with the model demonstrating up to a 88.9% accurate with the test set.
